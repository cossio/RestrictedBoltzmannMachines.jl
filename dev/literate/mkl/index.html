MKL vs. OpenBLAS

With an Intel CPU, MKL is generally faster than OpenBLAS. Let's do a quick comparison.

import MLDatasets
import Makie
import CairoMakie

Load MNIST

Float = Float32
train_x, train_y = MLDatasets.MNIST.traindata()
train_x = Array{Float}(train_x[:, :, train_y .== 0] .≥ 0.5)

Make sure we are using OpenBLAS first:

using LinearAlgebra
LinearAlgebra.__init__() # use OpenBLAS
BLAS.get_config()

import RestrictedBoltzmannMachines as RBMs
