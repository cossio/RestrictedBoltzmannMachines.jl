Gradients with Zygote

It is possible to calculate gradients with Zygote. Let's compare performance to explicit gradients.

import MKL
import MLDatasets
import Makie
import CairoMakie
import RestrictedBoltzmannMachines as RBMs

Setup

Float = Float32
epochs = 100
batchsize = 128

Load MNIST

train_x, train_y = MLDatasets.MNIST.traindata()
train_x = Array{Float}(train_x[:, :, train_y .== 0] .≥ 0.5)

Train using explicit gradients

rbm_∂s = RBMs.BinaryRBM(Float, (28,28), 128)
RBMs.initialize!(rbm_∂s, train_x)
history_∂s = RBMs.cd!(rbm_∂s, train_x; epochs, batchsize)
